# RouteSearch

한 줄 소개: 네이버 검색 SDK와 티맵 API를 활용한 경로 검색 애플리케이션 <p/>
팀 구성: 안드로이드 앱 개발 1명

## 🖥 Development

### Features
- 지도: 사용자가 컨트롤할 수 있는 지도 표시
- 경로 설정: 장소를 검색하여 출발지, 목적지 설정
- 길 찾기: 경로 표시
